# sitehome-i-resource

## 项目简介

sitehome-i-resource 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面彼此独立，不绑定特定域名或网站，适合作为静态资源存放或展示使用。

## 目录结构

```
sitehome-i-resource/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面引用的图片、样式、脚本等资源文件
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰便于识别。
- `assets/`：按需组织，避免跨页面资源冲突。

## 页面归档说明

- 每个页面均为完整的 HTML 文件，可直接在浏览器中打开。
- 页面之间无相互依赖关系，可单独使用或部署。
- 归档内容可能不定期更新或补充，不保证持续维护。

## 维护说明

- 欢迎通过 Issues 提交问题或建议。
- 如需贡献新页面，请遵循以下原则：
  - 页面应保持独立、自包含。
  - 避免引入外部不可控资源。
  - 提交前确保基本功能正常。
- 本仓库为个人或小组归档用途，不承担长期可用性保证。
